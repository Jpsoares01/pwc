Acessibilidade

A leitura de ecrã tem sido um pilar fundamental para a acessibilidade digital. É esta tecnologia que permite a pessoas com deficiência visual utilizarem computadores e smartphones — seja para lazer, trabalho ou apenas para se manterem informadas. Basicamente, o software traduz os elementos visuais e textuais do ecrã em informação auditiva ou tátil, garantindo uma navegação verdadeiramente autónoma.
